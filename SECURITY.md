# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 4.0.x   | :white_check_mark: |
| < 4.0   | :x:                |

## Reporting a Vulnerability

If you see a vulnerability, feel free to file an issue and/or a pull request,
and I'll review it to see if it's a sensible add to the repository.
